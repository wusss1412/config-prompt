Personal dotfiles, configuration notes, and rebuild prompts.

- `codex/my-global-config-prompt.md` - Codex-specific prompt for rebuilding and maintaining the user global configuration layer.

## Notes

- Keep secrets, tokens, SSH keys, machine-specific absolute paths, and private work configuration out of this public repository.
- Prefer portable descriptions, examples, and templates over direct copies of sensitive local configuration.
